# JavaScript - Warm up

This project is a first step into JavaScript on the command line using Node.js.

## Requirements

- All scripts run with Node.js 14.x on Ubuntu 20.04
- First line of every file: #!/usr/bin/node
- Code must follow semistandard style (Standard + semicolons)
- No use of \`var\`, only \`let\` and \`const\`
- All files must be executable

## What this project practices

- Printing text with \`console.log\`
- Declaring constants and variables (\`const\`, \`let\`)
- Basic control flow with \`if\`, \`else\`, loops
- Working with arguments from \`process.argv\`
- Simple data types and dictionaries in JavaScript
