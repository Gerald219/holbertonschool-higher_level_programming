# RESTful API — Holberton

**Goal:** Learn HTTP/HTTPS basics, use curl, consume APIs with Python, build simple APIs with `http.server` and Flask, and add basic auth/JWT.

## Project Structure
- `notes/` — study notes and quick cards
- `task_02_requests.py` — Python requests + CSV export
- `task_03_http_server.py` — simple API with http.server
- `task_04_flask.py` — simple API with Flask
- `task_05_basic_security.py` — basic auth + JWT
