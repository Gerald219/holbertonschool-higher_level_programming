# Python - Data Structures: Lists, Tuples

Covers how to work with lists, tuples, and strings using basic Python. Tasks involve printing values, accessing or changing elements by index, reversing lists, copying without modifying the original, removing specific characters from strings, formatting grids, combining tuples, checking types or values, and swapping variables.

Key tools and concepts used:
- Indexing, list traversal, slicing
- `if` statements and condition checks
- Loops and list comprehensions
- Tuple unpacking and arithmetic
- String handling and filtering
- Creating copies to avoid modifying original data
- Writing functions with clean logic and no imports
