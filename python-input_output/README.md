# Python - Input/Output

This project focuses on file operations and data serialization in Python.  
It covers reading and writing files, handling JSON data, and working with file systems using standard I/O functions and modules.

## Key Concepts
- Reading/writing files using `with`
- Working with file cursors
- `sys.argv` and command-line arguments
- JSON serialization (`json.dumps`) and deserialization (`json.loads`)
- Saving/loading Python objects to/from files
- File append mode vs overwrite
- Custom object serialization (e.g., class to JSON dict)
- Log parsing and handling standard input/output
