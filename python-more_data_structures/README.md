# python-more_data_structures

This project builds on basic Python skills by focusing on sets, dictionaries, and some light data transformations.

Each task is a short, focused function using only built-in Python tools — no imports.

Main topics:
- Finding common or different values in sets
- Updating, sorting, or deleting dictionary keys
- Using map() to modify lists or matrices
- Writing clean, readable logic with no side effects

The goal is to stay efficient and keep the original data unchanged unless required. All tasks follow pycodestyle and are tested for accuracy.
