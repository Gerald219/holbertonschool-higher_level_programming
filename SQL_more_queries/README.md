# SQL More Queries

This project contains a series of SQL tasks that help reinforce knowledge of databases, queries, and user management. 

## Tasks Overview

- **Task 0**: List privileges of MySQL users `user_0d_1` and `user_0d_2`.
- **Task 1**: Create the MySQL user `user_0d_1` with full privileges.
- **Task 2**: Create the database `hbtn_0d_2` and user `user_0d_2`, granting only `SELECT` privilege.
- **Task 3**: Create the `force_name` table with a non-null `name` field.
- **Task 4**: Create the `id_not_null` table with `id` having a default value of 1.
- **Task 5**: Create the `unique_id` table with `id` as a unique field.
- **Task 6**: Create the `states` table in the database `hbtn_0d_usa`.
- **Task 7**: Create the `cities` table in the database `hbtn_0d_usa` with a foreign key reference to `states`.
- **Task 8**: List all cities in California.
- **Task 9**: List all cities by states, showing `cities.id`, `cities.name`, and `states.name`.
- **Task 10**: List all shows with at least one genre linked.
- **Task 11**: List all shows, including those without a genre.
- **Task 12**: List all shows without a genre.
- **Task 13**: List the number of shows for each genre.
- **Task 14**: List all genres for the show `Dexter`.
- **Task 15**: List all shows in the genre `Comedy`.
- **Task 16**: List all shows and their genres.
- **Task 17**: List all genres not linked to the show `Dexter`.
- **Task 18**: List all shows that are not in the genre `Comedy`.
- **Task 19**: List all shows sorted by their rating.
- **Task 20**: List all genres sorted by their rating.
- **Task 21**: Write a blog post explaining SQL Database Engines.
