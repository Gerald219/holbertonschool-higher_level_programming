# Python - Import & Modules

These tasks focus on using imports and handling command-line arguments. The scripts work with basic math functions, arguments passed from the terminal, and reading values or names from modules.

Covers:
- Importing specific functions and variables from other files
- Using math functions like add, sub, mul, and div from a separate file
- Reading and printing arguments passed to a script
- Adding all arguments together using a loop
- Listing module contents, ignoring special names like __name__
- Grabbing a variable from a file and printing it

No external libraries are used. Everything is built with simple logic, formatted print statements, and tested using terminal commands.
